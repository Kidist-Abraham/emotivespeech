analysis.zero_crossing_rate_blocks(wavedata, Chunk_Size, fs)¶

This module computes the zero crossing rate of blocks. It is important in classifying/detecting the voiced and unvoiced region. If the ZCR is smaller then it is categorized as voiced. If the ZCR is greater then it is categorized as unvoiced.
